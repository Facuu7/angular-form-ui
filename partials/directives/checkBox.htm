checkBox Go to documentation

This directive gives developers a fancy way to present users with a yes or no choice.

Javascript

$scope.checked = true;

HTML

<check-box ng-model="booleanValue"><check-box>

Arrays

When working with arrays, check-box has you covered. As you check items an array will be populated with the appropriate values.


{{ favoriteColors }}

Javascript

$scope.colors = ['Blue', 'Orange', 'Yellow', 'Purple'];
$scope.favoriteColors = ['Orange'];

HTML

<div ng-repeat="color in colors">
    <label><check-box ng-model="favoriteColors" value="color"></check-box> {{ color }}</label>
</div>
