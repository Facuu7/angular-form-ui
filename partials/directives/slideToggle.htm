slideToggle Go to documentation

This directive gives developers a fancy way to present users with a yes or no choice.

HTML

<slide-toggle ng-model="slideToggleOnOff"><slide-toggle>

Custom Labels

Javascript

<slide-toggle on-label="Active" off-label="Inactive" ng-model="slideToggleActiveInactve"><slide-toggle>

State Triggers

Javascript

<slide-toggle on="onTrigger" off="offTrigger" ng-model="slideToggleTrigger" disable-triggers-on-init><slide-toggle>
