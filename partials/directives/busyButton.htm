busyButton Go to documentation

This directive provides 2 key elements every Angular application needs:

  • Communicates to the user that the application is busy
  • Prevents multiple calls to ngClick while application is busy

HTML

<button busy-button busy-text="Saving..." ng-click="save()">Click me!<button>

Javascript

$scope.save = function () {
    var deferred = $q.defer();
    $timeout(function () {
        //do something here
        deferred.resolve();
    }, 2000);
    return deferred.promise;
}
